# LinkedInOptimization

AI-powered LinkedIn Profile Optimizer that transforms resumes into optimized LinkedIn profiles.

## Overview

This project helps improve LinkedIn profile content using AI and prompt engineering. It is designed to enhance:

- Headlines
- About / Summary sections
- Experience descriptions
- Skills
- Recruiter visibility and ATS relevance

## Tech Stack

- Next.js
- React
- TypeScript
- AI / NLP prompt-based optimization
